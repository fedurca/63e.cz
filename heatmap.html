Leaflet se nepodařilo načíst.

Aplikace potřebuje Leaflet a leaflet-heat. Loader zkouší lokální soubory a pak CDN fallback.

Offline režim: dej vedle tohoto HTML buď:

  • leaflet.css, leaflet.js, leaflet-heat.js
  • nebo do lib/: lib/leaflet.css, lib/leaflet.js, lib/leaflet-heat.js

A pro automatické načítání targets_fixed_world.json používej http:// (např. python -m http.server).

Podrobnosti jsou v debug logu.
